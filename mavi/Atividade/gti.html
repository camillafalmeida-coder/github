Sobre git

O GitHub é uma plataforma para desenvolvedores que permite criar, armazenar, gerenciar e compartilhar seu código. Ela usa o Git para fornecer controle de versão distribuído, e o próprio GitHub fornece controle de acesso, rastreamento de bugs, solicitações de recursos de software, gerenciamento de tarefas, integração contínua e wikis para cada projeto. O GitHub, com sede em São Francisco, é operado pela GitHub, Inc., uma subsidiária da Microsoft desde 2018.
