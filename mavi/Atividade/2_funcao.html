A principal função do HTML (HTML) é estruturar o conteúdo de uma página web.

Em termos simples, ele serve para dizer ao navegador:

  • o que é um título
  • o que é um parágrafo
  • onde há imagens
  • quais partes são links
  • como o conteúdo está organizado

Sem HTML, a internet não teria uma “estrutura” — seria apenas texto solto sem organização.
Funções principais do HTML:
  • Estruturação: organiza o conteúdo (títulos, textos, listas, tabelas)
  • Semântica: dá significado aos elementos (ex: header, article, footer)
  • Hiperlinks: permite navegar entre páginas usando links
  • Integração: incorpora imagens, vídeos e outros recursos

Aqui o HTML está:
  • definindo um título (h1)
  • criando um texto (p)
  • adicionando um link (a)

👉 Resumindo: O HTML funciona como o “esqueleto” de um site. Ele define a estrutura, enquanto o design e a interatividade ficam por conta do CSS e do JavaScript.
