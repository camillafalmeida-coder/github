Descrição de GitHub

HTML (HTML) é a sigla para HyperText Markup Language, ou “Linguagem de Marcação de Hipertexto”. Ele é a base da construção de páginas na web.
Em vez de ser uma linguagem de programação tradicional, o HTML é uma linguagem de marcação, usada para estruturar o conteúdo de um site. Isso significa que ele organiza textos, imagens, links, vídeos e outros elementos por meio de **tags** (etiquetas).

Por exemplo:

H1 define um título
p define um parágrafo
a cria links
img insere imagens
  • A estrutura da página é definida
  • O conteúdo (título e texto) é organizado
  • O navegador interpreta o código e mostra a página visualmente
HTML geralmente trabalha junto com:
  • CSS (para estilo/visual)
  • JavaScript (para interatividade)
